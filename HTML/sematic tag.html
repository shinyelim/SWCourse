폰트를 두껍게

<b>태그는css처럼 단순하게 웹 브라우저에 표시되는 모양만 결정함 Css에서 폰트 두께는 font-weight속성으로 조절한다. 폰트를 두껍게

<strong>태그는 semantic tag로 HTML문서내에서 강조하고싶은 내용에사용한다.

둘 다 폰트가 두꺼워지지만 스토롱태그는 내용을 강조하는 시멘틱태그이다.
<igt;태그는 css처럼 웹 사이트에 표시되는 모양만 결정됨

<em>태그는 semantic tag로 html에서 이탤릭체로 내용을 강조하고싶을때 사용한다.

이탤릭체 내용 강조없이 모양만 이탤릭체로 표현하고 싶을때는 css font-style 속성을 가지고있다.

<s>태그는 화면상에 표시되는 모양 취소선

취소된 내용이라는 강조가 아닌 디자인적으로 중간 가로선을 굿게 만들싶을때 css- text-decoration 속성을 가진다.

<del>태그는 sematic tag로 문서내에서 삭제된 내용이라는걸 강조하고싶을때 사용한다. 이건 sematic tag!

<&ugt;태그는 웹 브라우저에 표시되는 모양만 결정 언더라인 css text-decoration:underline

<ins>태그는 sematic tag로 html문서에 새로 삽인된 내용이라는 걸 강조하고싶을때 사용한다.
